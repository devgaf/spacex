## Uso

Clonar el proyecto

```bash
  git clone https://github.com/devgaf/spacex.git
```

Ir al directorio del proyecto

```bash
  cd spacex
```

Instalar depndencias

```bash
  mvn clean install
```

Poner en marcha el servidor

```bash
  mvn spring-boot:run
```
